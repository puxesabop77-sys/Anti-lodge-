# 🚀 Anti Lodge Competition
**Home Away From Home** | *Bahadurganj, Bihar*

Welcome to the official digital ecosystem for the **Anti Lodge Competition**. This platform is a high-tech, browser-based challenge platform developed to engage students in a seamless, interactive experience.

## 🛠 Project Overview
Anti Lodge is a student housing and community project. This competition platform utilizes a lightweight, high-performance architecture allowing for secure authentication and personalized challenges without the need for complex backend APIs.

## 🔑 Key Features
*   **High-Tech Interface**: Modern, dark-mode terminal aesthetic designed for a seamless user experience.
*   **Secure Manual Authorization**: Custom generated ID/Password system to ensure controlled access.
*   **Personalized Challenge Desk**: Each user is served a specific question based on their authorization.
*   **Admin Command Center**: A centralized dashboard to manage users, generate credentials, and deploy questions instantly.

## 📂 Project Structure
*   `index.html`: The landing portal for participants to initiate their journey.
*   `login.html`: The secure entry point for authorized users.
*   `desk.html`: The active competition desk where participants face their challenge.
*   `admin.html`: The command center for the Anti Lodge administration team.
*   `style.css`: Global theme stylesheet featuring the signature Anti Lodge orange and navy palette.
*   `vercel.json`: Configuration file for clean routing and deployment.

## 🚀 Deployment Instructions
1.  **Clone the Repository**: Download or clone this project to your local machine.
2.  **Hosting**: This project is optimized for deployment on **Vercel**.
3.  **Clean Routing**: Ensure `vercel.json` is in your root folder for seamless URL navigation (e.g., `/login` instead of `/login.html`).

## 💻 Tech Stack
*   **Frontend**: HTML5, CSS3, JavaScript.
*   **Data Management**: Browser `localStorage` for lightweight, secure, and instant data persistence.
*   **Branding**: Inspired by the Anti Lodge mission of creating a technology-driven living space for students.
